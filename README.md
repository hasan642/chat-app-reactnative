# Chatsy
### Installation
```sh
$ git clone https://github.com/hasanal1995/chat-app-reactnative.git
$ cd chat-app-reactnative
$ yarn
$ cd ios && pod install
```
### Running for Android
```sh
$ yarn android
```
### Running for iOS
```sh
$ yarn ios
```
